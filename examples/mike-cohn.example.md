### Título
Login de usuário com e-mail e senha.

### Critérios de Aceitação
- [ ] Um usuário registrado pode fazer login com seu e-mail e senha.
- [ ] Um usuário não registrado não consegue fazer login.
- [ ] Se as credenciais estiverem incorretas, uma mensagem de erro é exibida.
- [ ] Se o login for bem-sucedido, o usuário é redirecionado para sua página de perfil.
- [ ] A senha não deve ser visível (deve estar mascarada).
